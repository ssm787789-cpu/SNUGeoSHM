# SNUGeoSHM

Digital Twin Platform for Offshore Wind Turbine Structural Health Monitoring

## Installation
```bash
pip install -r requirements.txt
```

## Run
```bash
cd frontend
python app.py
```

Open browser: http://127.0.0.1:8050

## Features

- Map Overlay
- 3D Soil Profile (GemPy)
- Foundation Springs (Optum GX)
- FE Simulation (OpenSeesPy)
- Analytics (Data Ingestion, SSI-COV, Anomaly Detection)
- Scenario Builder
